An exciting **SQL detective** project where you solve a real crime mystery using database queries.

## ✨ Overview

On the morning of July 28, 2023, a crime was committed in **Fiftyville**. You are given a SQLite database (`fiftyville.db`) containing crime scene reports, interviews, bank transactions, airport data, and flight records.

Your mission:  
**Find the thief, their accomplice, and the city they escaped to** — using only SQL queries.

## 🎯 Features

- ✅ Full forensic investigation using SQL
- ✅ Analyzes multiple tables: `crime_scene_report`, `interviews`, `people`, `bank_accounts`, `atm_transactions`, `flights`, `airports`, `passengers`
- ✅ Uses JOINs, subqueries, date filtering, and pattern matching
- ✅ Step-by-step logical deduction
- ✅ Clean and well-commented `.sql` files
- ✅ Real-world application of database skills
